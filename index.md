# Abstract Appsec

Most conversations about application security seem to revolve around specific technologies, 
which I find unfortunate.

This site is meant to be a casual-ish and somewhat approachable place for my soapboxes on
application security topics which usually do not revolve around specific SAST and DAST solutions, 
but rather the underlying ideas of what security even actually is.
